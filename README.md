# AlienPAK
A tool to preview, export, and re-import files within Alien: Isolation's PAK archives.

***This project is currently a work in progress!***

---

Currently supported is:
- PAK2 (ui/animations)
  - Open/parse
  - Export files
  
---

Eventually, the tool will support:
- PAK2 (ui/animations)
  - Import files
  - Preview files
  
- PAK (models/textures)
  - Open/parse
  - Export files
  - Preview files
  - Import files(?)
